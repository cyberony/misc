# Misc – Workspace memory

This repo is part of a multi-root workspace (with `elegance` and `sandbox`).

- **PDF extraction**: `extract-pdf-text.js` + `pdf-parse@1.1.1`. Run: `node extract-pdf-text.js <path-to-pdf>`. Output goes to `_extracted.txt` (same dir as PDF or `misc/_extracted.txt`).
- **Summaries**: Some papers/summaries live here (e.g. `ssrn-6097646-SUMMARY.md`). Prefer extracting PDF text then working from `_extracted.txt`.
