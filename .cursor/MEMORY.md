# Misc – Workspace memory

This repo is part of a multi-root workspace (with `elegance` and `sandbox`).

- **PDF extraction**: `extract-pdf-text.js` + `pdf-parse@1.1.1`. Run: `node extract-pdf-text.js <path-to-pdf>`. Output goes to `_extracted.txt` in the same directory as the PDF.
- **Stuffs**: Notes, PDFs, summaries (e.g. `stuffs/ssrn-6097646-SUMMARY.md`) under `stuffs/`. **`ai-compendium`** lives at repo root next to `stuffs/`. Prefer extracting PDF text then working from `_extracted.txt`.
