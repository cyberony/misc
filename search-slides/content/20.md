How  the  fringe  is  tracked  differentiates  between  BFS  and  DFS:
DFS  uses  a  stack
LIFO  (Last  in,  first  out)
BFS  uses  a  queue
FIFO  (First  in,  first  out)
The  first  one  in  will  be  the  first  one  out.
The  last  one  going  on  the  stack  will  be  the  first  one  out.
People  in  queue  at  the  polls
Plates  in  a  stack
