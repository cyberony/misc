Search  can  be  done  upfront  (given  complete  information  and  feasibility).
That  would  be  akin  to  planning  followed  by  execution.
Or  actions  can  be  interleaved  with  search.
Search  can  occur  in  a  geographical  space:  think  finding  a  route  from  point  A  to  point  B.
Search  can  also  be  fruitful  for  abstract  spaces  (e.g.,  concept  spaces).
Where  in  the  real  world  is  search  used?  In  a  variety  of  scenarios:
  Path  finding,  file  system  traversal,  social  network  connection  finding,  game  move  finding,  
chain-of-thought  exploration,  logistics,  planning  and  exploration  in  robotics,  and  much  more!
